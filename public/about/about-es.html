GeoSG

GeoSG es un geoportal de investigación y desarrollo destinado a la visualización, gestión y análisis de datos espaciales del área del municipio catastral de Slovenj Gradec. El sistema fue desarrollado en el marco de un trabajo de fin de máster como plataforma WebGIS demostrativa, que representa el resultado de la investigación de enfoques arquitectónicos, cartográficos y de datos modernos en el desarrollo de aplicaciones GIS web.
El geoportal combina la visualización cartográfica interactiva, la edición de entidades espaciales, la comunicación mediante REST API y la integración de tecnologías GIS de código abierto en una arquitectura modular unificada. El valor especial del sistema reside en la conexión de la visualización cartográfica, los datos de atributos y las operaciones espaciales en un entorno de trabajo único, que permite una gestión eficiente, transparente e intuitiva de la información espacial. Al igual que en el funcionamiento del geoportal colaboran numerosos componentes de la aplicación, todo el sistema se basa en la integración mutua de varias tecnologías de código abierto que juntas forman una solución WebGIS tecnológicamente moderna, escalable y potente.

Arquitectura tecnológica del geoportal

El sistema se basa en una arquitectura multinivel que incluye una interfaz de usuario Angular, una capa de aplicación Django REST, GeoServer para la distribución de datos espaciales y una base de datos espacial PostgreSQL/PostGIS. Los servicios individuales están organizados en contenedores Docker, lo que permite una clara separación de responsabilidades entre los componentes y su eficaz interconexión en un entorno WebGIS unificado y estable.

La comunicación entre las distintas partes del sistema se realiza a través de REST API y los servicios GIS estándar WMS y WFS, que permiten el intercambio, la distribución y la visualización eficientes de datos espaciales. La visualización cartográfica se basa en la biblioteca OpenLayers, que permite interacciones avanzadas con el mapa directamente en la interfaz de usuario.

La integración mutua de las tecnologías utilizadas permite el funcionamiento coordinado de la visualización cartográfica, el procesamiento de datos del lado del servidor y la base de datos espacial, lo que representa uno de los elementos clave de la funcionalidad del geoportal y permite el establecimiento de una solución WebGIS tecnológicamente moderna, escalable y potente.

Funcionalidades del geoportal

GeoSG permite la gestión integral de datos espaciales a través de un entorno GIS web interactivo que combina la visualización cartográfica, los datos de atributos y las herramientas de edición de entidades espaciales en una experiencia de usuario unificada.

El sistema admite la visualización, introducción, edición y eliminación de datos espaciales para distintos tipos de entidades, como parcelas, edificios, carreteras y direcciones. El usuario puede crear y editar geometrías directamente en el mapa mediante herramientas cartográficas interactivas.
El geoportal también permite la búsqueda avanzada, el filtrado y la localización de entidades espaciales, la sincronización de la visualización cartográfica con los datos de atributos y la visualización flexible de capas cartográficas. Todos los cambios en los datos espaciales introducidos o editados se almacenan permanentemente en una base de datos centralizada, lo que garantiza que la información del sistema esté actualizada y sea coherente. Más concretamente, el geoportal ofrece al usuario las siguientes funcionalidades principales:

  • exploración sencilla de datos espaciales mediante visualización cartográfica interactiva,
  • desplazamiento, zoom y navegación rápidos en el mapa,
  • dibujo interactivo, edición, marcado visual y eliminación de objetos espaciales directamente en el mapa sin necesidad de utilizar programas GIS de escritorio,
  • compatibilidad con el ajuste automático (snapping) a geometrías existentes al dibujar y editar objetos espaciales,
  • visualización de datos de atributos de objetos espaciales seleccionados en el mapa,
  • sincronización de la visualización cartográfica, las tablas y los formularios en un entorno de usuario unificado,
  • búsqueda avanzada, filtrado y localización rápida de parcelas, edificios, direcciones y otras entidades espaciales,
  • ajuste automático de la visualización de etiquetas de atributos de objetos en el mapa según la escala de visualización,
  • visualización flexible de capas cartográficas con la posibilidad de activar y desactivar vistas de contenido individuales,
  • visualización simultánea de diferentes capas cartográficas para facilitar el análisis espacial,
  • conmutación automática entre diferentes niveles de geometrías simplificadas de la capa WFS de parcelas según la escala de visualización del mapa actual,
  • almacenamiento permanente de los datos espaciales introducidos y editados por el usuario en una base de datos centralizada,
  • actualización automática de las capas cartográficas WMS de GeoSG en función de los cambios en los datos del sistema,
  • compatibilidad con los servicios GIS estándar WMS y WFS y posibilidad de utilizar las capas WMS de GeoSG creadas en otras aplicaciones GIS y sistemas de información,
  • importación y exportación de datos espaciales en formatos GIS estándar para su posterior uso en otros programas GIS,
  • acceso al sistema a través de un navegador web sin necesidad de instalar software adicional,
  • interfaz de usuario multilingüe, página web informativa del programa y manual de usuario con compatibilidad con los idiomas esloveno, inglés y español,
  • guardado automático del idioma de la interfaz de usuario seleccionado y del tema visual mediante el almacenamiento local del navegador (Local Storage),
  • adaptación automática del idioma de la página web informativa y del manual de usuario según el idioma seleccionado actualmente en el entorno de usuario,
  • página web informativa del programa y manual de usuario integrado, optimizados también para su uso en dispositivos móviles,
  • temas visuales adaptables para la interfaz de usuario y cambio de tamaño dinámico de las distintas partes de la interfaz de usuario,
  • visualización de las coordenadas de posición del cursor y ajuste dinámico de la escala gráfica según el nivel de zoom del mapa,
  • interfaz de usuario intuitiva para facilitar el trabajo incluso a usuarios menos experimentados,
  • posibilidad de uso básico del geoportal sin necesidad de iniciar sesión,
  • uso de cuentas de usuario e inicio de sesión para acceder a funcionalidades avanzadas del geoportal,
  • conservación del estado de inicio de sesión del usuario entre actualizaciones de página y visitas posteriores a la aplicación sin necesidad de volver a iniciar sesión,
  • funcionamiento coordinado del inicio de sesión del usuario en varias pestañas del mismo navegador web,
  • gestión de cuentas de usuario y derechos de acceso según el rol de cada usuario, con la posibilidad de ajustar fácilmente los permisos a través del entorno Django Admin por parte de las personas autorizadas.

El geoportal permite así un trabajo eficiente tanto con datos de atributos como con datos geométricos, y representa una solución GIS web moderna para la gestión y visualización de información espacial.

Aspecto investigador del geoportal

Como ya se ha mencionado, GeoSG no fue desarrollado únicamente como un sistema GIS web demostrativo, sino que también tiene valor investigador, ya que aborda y analiza diferentes enfoques para el desarrollo de soluciones WebGIS modernas. La investigación se orientó, entre otras cosas, a la comparación de diferentes enfoques arquitectónicos, la conexión de tecnologías GIS de código abierto y su integración en un entorno GIS web unificado, modular y escalable.

En el trabajo de fin de máster se hizo especial hincapié en la investigación de la simplificación de geometrías espaciales con el objetivo de mejorar el rendimiento, la claridad y la capacidad de respuesta de la visualización cartográfica cuando se trabaja con grandes cantidades de datos espaciales. Además, el componente investigador incluyó una comparación de diferentes enfoques para implementar servicios Django REST API y un análisis de su idoneidad para el desarrollo de sistemas GIS web modernos.

Los resultados de la investigación están directamente incorporados en el funcionamiento del geoportal y son parcialmente visibles para el usuario a través de las capas cartográficas especializadas "Parcelas – análisis de simplificaciones" y "Parcelas – visualización LOD automática", que muestran diferentes niveles de geometrías de parcelas simplificadas según la escala de visualización. El aspecto investigador de la implementación del servicio Django REST API se presenta adicionalmente a través del acceso a la documentación de la API Swagger y Redoc, disponible para el usuario en el manual de usuario bajo la pestaña FAQ en la sección "Enlaces importantes". Este acceso directo a la documentación de la API de desarrollo en geoportales de producción no es habitual, pero en GeoSG se ha habilitado intencionalmente debido al carácter investigador y demostrativo del sistema.
