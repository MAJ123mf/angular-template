WEB DEVELOP AND GEOPORTALS

This is a simple web application for managing land registry data.
It allows users to insert, update, and delete records related to parcels, addresses, and roads.
The application is built using Angular and integrates with a PostgreSQL database.
In application ve use leaflet map with OpenStreetMap tiles to input the data.
Back-end is implemented using Django.
The application is designed to be user-friendly and provides a responsive interface for easy navigation.
